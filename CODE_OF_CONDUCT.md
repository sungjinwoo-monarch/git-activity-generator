# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, inclusive, and harassment-free environment for everyone.

## Expected Behavior

* Be respectful
* Be constructive
* Accept feedback professionally
* Support newcomers
* Focus on collaboration

## Unacceptable Behavior

* Harassment
* Personal attacks
* Discrimination
* Offensive language
* Trolling or disruptive behavior

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to:

* Issues
* Pull Requests
* Discussions
* Community interactions

By participating, you agree to follow these guidelines.
