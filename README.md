# 🎨 Git Activity Generator

Create beautiful GitHub contribution graph patterns through an interactive visual designer.

Git Activity Generator helps developers design custom contribution layouts, preview them in real time, and generate commands that reproduce those patterns on GitHub contribution graphs.

---

## ✨ Features

* Interactive contribution grid editor
* Real-time activity preview
* Custom pattern creation
* Git command generation
* Modern Japanese-inspired UI
* Responsive design
* Built with React + TypeScript + Vite
* Fast and lightweight

---

## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* npm
* Gemini API Key

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/git-activity-generator.git
```

Move into the project directory:

```bash
cd git-activity-generator
```

Install dependencies:

```bash
npm install
```

Create an environment file:

```env
GEMINI_API_KEY=your_api_key_here
```

Start development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
